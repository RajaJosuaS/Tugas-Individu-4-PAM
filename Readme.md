# Tugas Individu 4 PAM Redux Thunk & Saga

## Cara menginstall aplikasi (dalam mode pengembangan)
1. install module
```
npm install
```

2. install expo
```
npm start
```

3. build to apk ( android )
```sh 
eas build --platform android
```

## Link Aplikasi
